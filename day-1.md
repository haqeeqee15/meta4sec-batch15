Basic Command Linux

root = admin

Directory = Folder
Tab bisa dipakai sebagai autocomplete
Cara save nano = ctrl x + y + enter
Ctrl c = Untuk memberhentikan suatu proses

 1. sudo su = Untuk menjadi admin/super user
 2. ls = list directory / melihat isi folder dan file
 3. cd = change directory / untuk pindah folder
 4. cd .. = keluar dari suatu directory
 5. clear = menghapus history terminal
 6. mkdir = membuat directory
 7. rmdir = menghapus directory
 8. nano = membuat file / mengedit file
 9. cat = membuat file / mengedit file
 10. rm = menghapus file
 11. cp = copy file
 12. pwd = melihat lokasi user
 13. mv = move file / rename file
 14. chmod = mengubah hak akses suatu file

permission = hak akses
read - write - execute

chmod 444   = read only
chmod 666   = read + write
chmod 777   = read + write + execute
